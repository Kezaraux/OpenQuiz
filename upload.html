Uploading your own Course

This is where you are able to upload new courses to the catalogue, for other people to enjoy. Simply select a valid json file, and then click upload.


Proper Course Guideline

This is a course json example 

{
    "title" : "Example Course",
    "description" : "This is the description for the example course.",
    "author" : "Author Name",
    "filters" : ["Example", "Filter"],   // View our list of filters for proper tagging
    "difficulty" : "Easy",
    "tests" : [
        {
            "title" : "Test 1",
            "course_title" : "Example Course",
            "description" : "This describes what the test will be on, and any other information you need to provide.",
            "testNumber" : 1,  // This is the number to determine the order of tests
            "questions" : [
                {
                    "title" : "What is 2 + 2?",
                    "choices" : ["1", "2", "3", "4"],
                    "answer" : 3   // This is the index in the array of the answer
                },
                {
                    "title" : "What is 2 x 2?",
                    "choices" : ["2", "4", "6", "8"],
                    "answer" : 1
                }
            ]
        }
    ]
}
